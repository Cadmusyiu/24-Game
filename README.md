# 24 Game

> 🎮 **Play live**: https://cadmusyiu.github.io/24-Game/

An arithmetic card puzzle: combine four dealt cards using `+ − × ÷` and parentheses to reach **24**. Ships with a built-in solver and a practice mode.

## Features
- Classic deal-four-cards puzzle with standard 24-game rules
- **Solver** — finds a valid expression when you're stuck
- **Practice mode** — drill mental arithmetic at your own pace
- Runs entirely in the browser, no build step

## How to run
Open `index.html` directly in a modern browser, or use the live demo above.

## Tech
Vanilla HTML / CSS / JavaScript — no dependencies.
